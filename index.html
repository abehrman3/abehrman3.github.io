Python OpenWeatherMap API Project

Using the power of data to identify weather trends...

For my first Python API project, I set out to determine if and how the weather changes as one approaches the equator. To start my weather analysis, I sourced the weather data of just over 500 random cities located across the globe, using the OpenWeatherMap API.

Once I had the dataset assembled, I visualized my the change in certain weather characterstics vs. the lattitude of my datapoints. The key weather factors I identified were temperature, cloudiness, wind speed, and humidity. Descriptions and explanations of my findings are hosted on this site. Although the you might intuitively guess many of the trends, it was interesting to see the power of data to objectively visualize what we might believe to be true.
